# FlowerRecognition
Implementacija projekta je data u obliku IPython notebook fajla. Projekat je implementiran koristeći Anaconda okruženje i Jupzter notebook IDE. Projekat je moguće pokrenuti direktno sa GitHub sajta, otvaranjem fajla: FlowerRecognition.ipynb. Za lokalno pokretanje projekta potrebno je instalirati Anaconda okruženje sa Jupyter alatom. Klonirati repozitorijum i pomoću terminala pozicionirati se u folder koji sadrži fajl FlowerRecognition.ipynb. Potom u terminalu unijeti komandu: jupyter notebook. Nakon pokretanja Jupyter servera u browseru se otvara prikaz struktre projekta. Kliknuti na link FlowerRecognition.ipynb. Za pokretanje otvorenog notebook fajla izabrati opcije Kernel -> Restart and run all i potvrditi akciju sa Restart and run all cells. Na stranici će biti prikazani kako izvorni kod sa pratećim obješnjenjima tako i rezultati izvršavanja koda.
